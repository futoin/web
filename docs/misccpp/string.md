# `futoin::string`

`IMemPool`-aware instances of `std::basic_string<>`:

- `futoin::string` with `char` type
- `futoin::u16string` with `char16_t` type
- `futoin::u32string` with `char32_t` type

## Usage

```cpp
#include <futoin/string.hpp>

void usage_example() {
    futoin::string abc{"abc"};
}
```
